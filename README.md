 # Bus Pirate - V3 #
 
 El "Bus Pirate" es una multi-herramienta, de código abierto, con la que podrás comunicarte con aparatos electrónicos. Posee un montón de característica que un intrépido hacker podría necesitar para el prototipado de sus próximos proyectos.
